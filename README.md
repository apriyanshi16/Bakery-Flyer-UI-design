
# 🍞 Bakery UI Design

## 📌 Project Overview

This project showcases a **Bakery Promotional UI Design** created to present fresh bakery products in an attractive and modern visual layout.
The goal of this design is to create a warm, appetizing, and user-friendly interface that attracts customers and highlights bakery specialties.

---

## ✨ Design Concept

* Warm bakery theme with brown & cream color palette
* Food-focused visual hierarchy
* Modern typography with clean layout
* Product highlights with pricing cards
* Promotional landing screen design

---

## 🎯 Objectives

* Improve product visibility
* Create engaging food promotion layout
* Maintain aesthetic and readable UI
* Deliver visually appealing customer experience
